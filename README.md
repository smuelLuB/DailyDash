# Daily Dashboard

ADHD daily planner — habits, top 3, MIT, deep work, shutdown ritual.

## Deploy to GitHub Pages

1. Create a new **public** repo on GitHub (e.g. `daily-dashboard`)
2. Upload `index.html` and `manifest.json` to the repo root
3. Go to **Settings → Pages**
4. Under *Source*, select **Deploy from a branch**
5. Choose **main** branch, **/ (root)** folder → click Save
6. Your site will be live at `https://YOUR-USERNAME.github.io/daily-dashboard/`

## Add to Android home screen

1. Open the URL in Chrome on your Android phone
2. Tap the **⋮ menu** (top right)
3. Tap **"Add to Home screen"**
4. Give it a name and tap Add
5. It will appear on your home screen and open fullscreen like an app

## Optional: icons

Add `icon-192.png` (192×192px) and `icon-512.png` (512×512px) to the repo
for a proper home screen icon. Without them it will use a screenshot as the icon.

## Data

All data is saved in your browser's localStorage — nothing leaves your phone.
Notes carry over day to day. Everything else resets at midnight.
