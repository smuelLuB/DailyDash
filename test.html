Run from a local server, not by double-clicking this file — open a terminal in this folder and run python3 -m http.server 8000 --bind 127.0.0.1 (the --bind flag keeps the server reachable only from this machine, not the rest of your network), then visit http://localhost:8000/test.html. Opening via file:// directly will not work: browsers block a test page from reaching into the app's iframe under that protocol.
